# AuctionPlatform

An auction platform where you can:
1. Register a Company
2. Register a User

(As a company):
1. Add Items to a Company catalog
2. create an Auction as a Company
3. Add offers to an Auction
4. Publish the Auction
5. Start an Auction
6. End an Auction

(As a User):
1. Place bids

An auction can be of 2 types:
-English auction
-Second bid auction

see "add link".
